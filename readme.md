# Crowdsource

Visit the production site: [Crowdsource](https://murmuring-temple-89463.herokuapp.com/)

To install locally:

1. Clone Repo
2. Cd into directory
3. Run `npm install`
4. Fire up development server - `npm start`
5. Visit [](http://localhost:3000/)

![](http://g.recordit.co/WF5cdqbJ4t.gif)

To run tests:

1. Clone Repo
2. Cd into directory
3. Run `npm install`
4. Run - `npm rest`
